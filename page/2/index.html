swift-6.24-tableView-1

发表于 2017-06-24

1、nil的用法
override func tableView(_ tableView: UITableView,
willSelectRowAt indexPath: IndexPath) -> IndexPath? {
return nil
}

-> 代表返回值的类型，可以是int、float、object等，也可以是nil，代表nothing
-> IndexPath? 代表返回值可能有，可能没有，可能是nil等
此句代表，该row在user tap的时候，不会被选中。

如果想要被选择的cell不会变成gray，可以采用上面的办法，也可以在tableview的属性里面selection—>none

2、IBOutlet的用法
@IBOutlet weak var textField: UITextField!
目的：输出用户在textfield的文字
方法（新）：点开assistant editor，ctrl+drag即可。
