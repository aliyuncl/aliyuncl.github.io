1、AppDelegate为整个应用的一个代理，提供程序启动、退出等类似监控的接口，所以也是一个object
所有的object都有功能和data，对于每个object，提供功能的方法就叫做method
2、当一个object的method完成以后，就会回到当时它离开的地方。

UIAlertController学习：

@IBAction func showAlert(){
let alertController = UIAlertController(title: “a”,message: “a”,preferredStyle: .alert)

let alertAction = UIAlertAction(title: “取消”,style: .cancel,handler:nil)
alertController.addAction(alertAction)
self.present(alertController,animated: true,completion: nil)

@IBAction func showAlert(){} 是指创建一个IB与viewcontroller的联系，IBAction接func，IBOutlet接 变量，一般是weak

警示框主要包括controller和action，分别用常量定义，然后再利用addAction的方法加入到controller，组成一个alert

let alertController = UIAlertController(title: “a”,message: “a”,preferredStyle: .alert)
属性：title，message，preferredStyle（包括两种.alert，弹出对话框，.actionSheet,下部弹出对话框）

let alertAction = UIAlertAction(title: “取消”,style: .cancel,handler:nil)
属性:title，style（包括常见的三种，cancel,default,destructure红色），handler:nil //handler:nil

alertController.addAction(alertAction)
对象.方法（参数），把action传入controller
然后self.present。。。。。

注意：
1、具体展示的位置看传入的顺序
2、.alert超过2个action，就会变成上下模型，而不会是左右模型

弹窗弹出输入框：
alertController.addTextField{(textField: UITextField!) -> Void in
textField.placeholder = “your name:”
}
alertController.addTextField{(textField: UITextField!) -> Void in
textField.placeholder = “password”
textField.isSecureTextEntry = true
}

2秒钟以后关闭对话框
DispatchQueue.main.asyncAfter(deadline: DispatchTime.now() + 2){
self.presentedViewController?.dismiss(animated: false, completion: nil)
}

var 变量名: Int = 0 // 定义一个变量，且指定这个变量的类型是Int，初始值是0
注意点：当method带有参数的时候，IB与view controller连接的时候，都会有一个：
@IBAction func sliderMoved(_ slider: UISlider){
currentValue = lroundf(slider.value)
}

1、(slider.value)，把其他变量转化为字符串形式
2、(_ slider: UISlider) 传入slider这样一个参数，类型是UISlider
3、lroundf“取整”，将浮点数变成Int类型。
