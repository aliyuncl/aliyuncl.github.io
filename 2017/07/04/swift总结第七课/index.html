swift总结第七课

发表于 2017-07-04 | |

背单词：

hierachy：层次、层系

navigation controller

tableview：

teminology：专有名词

toggle：切换

anatomy：剖析

span：跨度

cosmetic：表面的

TableView：

table：在cell里面展示数据。

cell：与row有关，但不相同。cell是一个视图（view），这个view可以展示一排（row）的数据。

通俗理解：

cell是展示一排一排数据的载体，row可能有很多，但cell就是屏幕展示的那些。

checkmark是一种tableviewcell內建的view（accessory）

reuse identifier：重用标识符，当滚动屏幕的时候，tableview将会去找空闲的cell去复用，以用于展示新的row。

每一个cell，都可以有自己的重用标识符的类型，可以是image、button、label等，所以实际应用环境中，我们需要设定不同cell的类型，以便可以在滚动的时候，使用正确的cell类型来复用。
