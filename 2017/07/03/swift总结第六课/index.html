Typora常见快捷键和技巧

command+T 插入表格

图片拖拽：Preferences -> Editor，Image Drag &Drop，默认不支持，需要手工设置。

支持实时目录：command+shift+b

删除线：~删除文本~ 删除文本

下划线：Underline Underline

插入表情：使用:happy:输入高兴的表情，使用:sad:输入悲伤的表情，使用:cry:输入哭的表情

下标：H~2~O

上标：y^2^=4

代码：

1、printg()，会产生printf()样式

2、输入~~~或者```然后回车，可以输入代码块

强调：**

使用两个*号强调内容（推荐使用）

使用两个下划线强调内容__

插入图片：直接拖拽即可。

插入URL连接：www.baidu.com，将会产生一个可点击的链接。< ..>

水平分割线：*或者—，然后回车，会生成一个大的水平分割线
