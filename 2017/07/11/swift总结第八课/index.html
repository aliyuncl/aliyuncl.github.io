swift总结第八课

发表于 2017-07-11 | |

单词：

computation 计算，估计

prototype 原型、图形、蓝本

spontaneously：自然地，自发地

pitfall 陷阱

cumbersome：笨重的，麻烦的

foolproof:万无一失的，肯定成功的

interchangeably:可替换的，可交替的

modal：模式的; 情态的; 形式的; 语气的;

概念：

reuse identifier：重用标识

cell reuse identifier：cell重用标识，允许在滚动过程中重复利用之前cell，而且还可以区别不同cell的types，以便正确利用类型并使用。

indexPath：指向table中的某一行，当tableview找view controller要cell的时候，必须通过行号来区别，也就是indexPath.row属性

拓展：经常看到table里面有些内容是分组的，一般以A、B结尾，可以使用indexPath.section属性来区分。

每个屏幕最多14个cell，可见的只有13个，还有一个scroll过程中出现。

if let 语法：

​ if let cell = tableView.cellForRow(at: indexPath){

​ if cell.accessoryType == .none {

​ cell.accessoryType = .checkmark

​ }else {

​ cell.accessoryType = .none

​ }

​ }

if let a = b 等号右边一定是个可选变量，如果有值则b赋值给a，当条件是true的时候执行语句，如果没值，那么直接执行else

tableview delegate

在swift中，声明类的时候，每个变量都需要有值，如果没有值的话，必须要给一个intializer method
